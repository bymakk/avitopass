# AvitoPass

Мониторинг Avito в Telegram.
Кидаешь ссылку поиска. Бот смотрит выдачу и пишет в чат, когда выходит свежее объявление.
Мой парсер не долбится спамом запросов на Avito: стабильно спокойно запрашивает страницы с кодом 200. Никаких 403, без 429, без SPFA.
Даю бота на тест: со всеми отчётами и затратами.
Писать сюда: [@by_mak](https://t.me/by_mak)

---

## Сколько поисков на IP

Для мобильных безлимитных прокси рабочая норма:

```
1 IP  →  5 поисков  →  опрос ~раз в 20 сек
```

Дальше линейно 1 к 5. Можно запустить исходя из логики 1 к 5, чтобы если коэфицент ниже, опрос шел раз в 10 сек, при увеличении нагрузки/подписок задержка увеличивалась - на ваше усмотрение. Но вообще можно и 1 к 6/7...

## Прокси

Для постоянной работы нужен мобильный безлимит. Страницы Avito тяжёлые, тариф «за гигабайты» на дистанции значительно дороже.
Ориентир по рынку: около **1 500 ₽** за IP в месяц на мобильный безлимит, или брать оптом по промокодам.
