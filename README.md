## Ёмкость

Ориентир для **мобильных безлимитных** прокси:

### **1 IP ≈ 5 поисков · обновление раз в ~20 секунд**

Больше адресов — больше поисков в той же пропорции. На один IP соотношение не «улучшается» само по себе: упираетесь в канал и очередь запросов.

Реже опрос — можно чуть плотнее на тот же IP. Гнать быстрее ~20 сек на пачке с одного адреса обычно бессмысленно.

**Резидентские** прокси обычно держат нагрузку хуже: на тот же объём поисков закладывайте примерно **в 2–3 раза** больше IP.

## Прокси

Для постоянной работы берите **мобильный безлимит**.

Выдача Avito тяжёлая по трафику. Тариф «за гигабайты» на дистанции почти всегда проигрывает безлимиту.

| | |
| :--- | :--- |
| **Мобильный безлимит** | Нормальный режим. Ориентир рынка ≈ **1 500 ₽ / IP** в месяц |
| **Резидентские** | Если мобильных нет. Нужно больше адресов на тот же объём |
| **По гигабайтам** | Ок для пары своих ссылок. На потоке обычно дорого |

Качество IP важнее любых «волшебных» обещаний: слабый канал режет стабильность быстрее, чем тонкая настройка бота.
