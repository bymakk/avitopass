# AvitoPass

Мониторинг Avito в Telegram.

Кидаешь ссылку поиска. Бот смотрит выдачу и пишет в чат, когда выходит свежее объявление. Без F5.

## Что делает

Пока вручную обновляешь выдачу, нормальные объявления уже разбирают.

AvitoPass держит твои поиски и присылает только свежие объявления в Telegram. Можно повесить несколько выдач сразу: телефоны, авто, квартиры и что ещё нужно.

Бот работает сам. Ты занимаешься сделками, а не обновлением вкладки.

## Сколько поисков вытянет пул

Для мобильных безлимитных прокси рабочая норма такая:

**1 IP = 5 поисков, опрос примерно раз в 20 секунд.**

Дальше линейно: сколько IP, столько и пятёрок поисков при том же интервале. На одном адресе соотношение само по себе не растёт. Упираешься в канал и в то, что запросы с одного IP идут по очереди.

Если опрашивать реже (30–60 сек), на тот же IP иногда влезает чуть больше. Гнать быстрее 20 сек на пачке с одного адреса обычно нет смысла.

Резидентские прокси держат нагрузку хуже. На тот же объём поисков закладывай примерно в 2–3 раза больше IP.

## Прокси

Для постоянной работы нужен мобильный безлимит. Страницы Avito тяжёлые, тариф «за гигабайты» на дистанции значительно дороже.

Ориентир по рынку: около **1 500 ₽** за IP в месяц на мобильный безлимит, или брать оптом по промокодам.
