# AvitoPass

Парсер объявлений и всего чего угодно на Avito в Telegram.
Бот смотрит выдачу и пишет в чат, когда выходит свежее объявление, может парсить объявления и любую другую информацию.
Мой парсер не долбится спамом запросов на Avito: стабильно спокойно запрашивает страницы с кодом 200 (никаких постоянных 403, без 429 и т д, без SPFA кукисов).
Даю бота на тест: со всеми отчётами о работае и сопутствующим.
Писать сюда: [@by_mak](https://t.me/by_mak)

## Сколько поисков на IP

Для мобильных безлимитных прокси рабочая норма:

```
1 IP  →  5 поисков  →  опрос ~раз в 20 сек
```

Можно запустить исходя из логики 1 к 5, если коэфицент ниже, объявления запрашиваются быстрее, в режиме 1 к 5 раз в 20 секунд.

Для серверных безлимитных прокси рабочая норма:

```
1 IP  →  2 поисков  →  опрос ~раз в 20-30 сек
```
